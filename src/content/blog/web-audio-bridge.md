音乐控件不应该一直压在页面上。新的舰桥广播只在顶部按钮展开时出现，平时收回到导航栏里，阅读文章时不会抢位置。

音频本身使用浏览器 Web Audio API 合成，不依赖外部音频文件。这样体积更小，音量、滤波、节奏和轨道切换都能通过原生 JS 实时控制。

## 音量问题

之前即使把控件拉满，实际听感仍然偏小。新版本把面板显示范围提高到 500%，内部通过主增益和压缩器处理，避免突然爆音，同时让最大音量真正有变化。

因为浏览器限制自动播放，用户仍然需要点击一次启动按钮。这个限制不能绕过，但可以把启动后的控制体验做顺。

## 三种声道

`Warp Drive` 偏低频推进感，适合首页浏览。

`Deep Relay` 偏深空信号，适合阅读长文。

`Signal Cascade` 偏明亮脉冲，适合切换档案时的节奏感。
