个人站的项目区不能只写项目名字。真正有用的项目展示需要让访问者快速知道：这个项目是什么、完成到什么程度、用了什么技术、可以从哪里继续看。

所以新的项目卡片加入了状态、年份、进度条、指标、技术栈和入口。它更像一个作品集模块，而不是普通博客链接。

## 项目卡片结构

顶部是状态和年份，告诉读者它是已上线、迭代中还是新功能。

中间是标题、副标题和说明，负责解释项目价值。

底部是指标、技术栈和入口，方便继续阅读相关文章或打开页面。

## 后续计划

以后如果有真实项目 Demo，可以给每个项目增加独立详情页、截图、功能清单和 GitHub 仓库链接。
